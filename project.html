Projects

Here are some projects I've done while taking my courses at the University of Oklahoma!

Pokédex

In my Non-Procedural Programming class, I built a Pokédex from the Pokémon series. It allows the user to pick a Pokémon from a list and it displays information about the Pokémon.

Tic-Tac-Toe

In my Non-Procedural Programming class, I created a Tic-Tac-Toe game using Windows Presentation Foundation (WPF). The user can play Tic-Tac-Toe, and it even keeps track of how many times each side has won.

Elysian Database Project

In my Database Management class, my group and I created a database to keep track of a fictional company's data. Over 15 queries were also created to keep track of various information that the client wanted such as top sales for an item per month or how much a product sold out.
